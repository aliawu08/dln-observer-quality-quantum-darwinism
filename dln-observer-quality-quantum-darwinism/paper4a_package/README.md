# Paper 4A package (manuscript + reproducibility scripts + tests)

## Directory layout

- `paper/`
  - `paper4a_pra_reviewed.tex` — manuscript (editorial/rigor pass aligned to PRA expectations)
  - `paper4a_observer_quality_major_revision.bib` — **placeholder** BibTeX entries (replace with full bibliographic data)
- `scripts/`
  - `central_spin_reproduce.py` — generates the central-spin redundancy figure and CSV
  - `reproduce_results_ABC.py` — reproduces the numerical values quoted in the manuscript (Results A–C)
- `tests/`
  - `run_tests.py` — deterministic checks for key inequalities and the central-spin numerics
- `figures/`
  - `central_spin_redundancy_vs_time.png` — generated by `scripts/central_spin_reproduce.py`
  - `central_spin_redundancy_vs_time.csv` — generated by `scripts/central_spin_reproduce.py`

## Quick start

### 1) Generate the central-spin figure

From the package root:

```bash
python scripts/central_spin_reproduce.py
```

This writes:

- `figures/central_spin_redundancy_vs_time.png`
- `figures/central_spin_redundancy_vs_time.csv`

### 2) Reproduce the numerical values quoted in the text (Results A–C)

```bash
python scripts/reproduce_results_ABC.py
```

### 3) Run tests

```bash
python tests/run_tests.py
```

## Notes

- The `.bib` file is intentionally minimal and contains placeholder entries to keep the package self-contained.
  Replace each entry with the final bibliographic metadata before submission.
- The reported numerical values are deterministic under `seed=2` and match the manuscript's "Numerical verification" section.
